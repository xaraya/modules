Installation
---------------
1. Extract zipped files under /modules directory (/Netquery subdirectory).
2. Install as usual from the modules page of the Xaraya admin panel.
3. Open Netquery admin and select module features to be available to users.
4. IMPORTANT: The ping and traceroute features require read/execute
   permission for your system's ping and traceroute executables.

That's it -- enjoy!
